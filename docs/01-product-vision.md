# 💕 Together

> La aplicación inteligente para administrar las finanzas de una pareja.

Versión del documento: 1.0

---

# 1. Visión del producto

Together nace con un objetivo simple:

> Ayudar a las parejas a tener una relación financiera saludable.

Actualmente existen aplicaciones para dividir gastos como Splitwise, pero ninguna realmente permite administrar las finanzas personales y compartidas como una sola experiencia.

Together busca convertirse en el espacio financiero de la pareja.

No es una billetera digital.

No almacena dinero.

No reemplaza Nequi ni los bancos.

Es un sistema inteligente que permite organizar, analizar y proyectar las finanzas de dos personas.

---

# 2. Problema

Muchas parejas presentan problemas como:

- No saben cuánto gasta cada uno.
- No saben quién ha aportado más.
- Pierden el control de las metas.
- No existe transparencia.
- Se olvidan de pagos.
- No tienen estadísticas.
- No saben cuánto pueden ahorrar.
- No conocen hábitos de consumo.

Esto genera discusiones y una mala administración del dinero.

---

# 3. Solución

Together centraliza toda la información financiera de una pareja.

Cada usuario posee:

- Finanzas privadas.
- Finanzas compartidas.
- Metas.
- Presupuestos.
- Estadísticas.
- IA financiera.

Todo dentro de una única aplicación.

---

# 4. Filosofía

El dinero continúa en las cuentas bancarias de los usuarios.

Together únicamente administra la información.

Esto permite:

- Mayor seguridad.
- Menor complejidad legal.
- Desarrollo más rápido.
- Integración futura con diferentes bancos.
