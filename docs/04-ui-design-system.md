# Filosofía de Diseño

Together debe sentirse como una aplicación Premium.

El usuario debe experimentar:

- tranquilidad
- organización
- elegancia
- confianza
- cercanía

El diseño está inspirado en el mockup original creado para Together.

No debe copiarse exactamente, pero sí mantener su identidad visual.

Inspiraciones:

- Apple Wallet
- Revolut
- Linear
- Notion
- Monzo

# Estados Vacíos

Nunca mostrar una pantalla vacía.

Siempre incluir:

Ilustración

Mensaje motivador

Botón de acción

Ejemplo:

"Aún no han registrado gastos compartidos ❤️"

---

# Tonalidad

Toda la aplicación debe hablar como un compañero financiero.

Nunca usar mensajes negativos.

Ejemplo

❌ Error.

✅ Parece que ocurrió un problema. Intentemos nuevamente.

---

# Principio de Diseño

Cada pantalla debe responder a estas preguntas:

¿Es elegante?

¿Es clara?

¿Puede entenderse en menos de 5 segundos?

¿Respeta el mockup original?

¿Se siente como una aplicación premium?

Si alguna respuesta es NO, el diseño deberá ajustarse antes de implementarse.
